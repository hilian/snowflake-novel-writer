# 雪花写作法小说创作助手

## 什么是雪花写作法

雪花写作法（Snowflake Method）是由Randy Ingermanson博士开发的一种结构化小说创作方法。它的核心思想是从一个简单的创意开始，像雪花一样逐渐扩展和复杂化，最终形成完整的小说。

## 雪花写作法的八个步骤

### 步骤1：一句话概括你的故事

**操作指南：**
- 用一句话概括你的整个故事
- 包含主角、目标、冲突和灾难
- 示例："一位年轻的魔法师必须在被黑暗势力摧毁世界之前，找到传说中的光明之石。"

**示例输出：**
```
一句话故事："一位失去记忆的赏金猎人必须在被追杀的同时，揭开自己神秘的过去。"
```

### 步骤2：扩展为一段摘要

**操作指南：**
- 将一句话扩展为一段（约5句话）的摘要
- 包含故事的开始、发展、高潮和结局
- 明确主角的动机和面临的挑战

**示例输出：**
```
故事摘要："在一个被军阀统治的未来世界，失去记忆的赏金猎人凯德接下了一个看似简单的任务——寻找一位失踪的科学家。但随着调查的深入，他发现这个任务与自己的过去息息相关。当他逐渐恢复记忆时，他意识到自己曾经是军阀的顶级杀手，而他现在必须在保护科学家的同时，面对自己黑暗的过去和前来追杀他的昔日同僚。最终，他必须做出选择：是继续逃避过去，还是直面自己的罪行并寻求救赎。"
```

### 步骤3：为主要角色创建简介

**操作指南：**
- 为每个主要角色创建简短的简介
- 包含角色的姓名、年龄、职业、外貌特征、性格特点和动机
- 至少包括主角、反派和重要配角

**示例输出：**
```
角色简介：

1. 凯德（主角）：35岁，赏金猎人，身手敏捷，擅长各种武器。外貌特征：伤疤纵横的脸，灰色眼睛，短发。性格：沉默寡言，行动果断，内心隐藏着痛苦的过去。动机：寻找自己失去的记忆，保护无辜的人。

2. 艾莉亚（重要配角）：28岁，科学家，天才发明家。外貌特征：金色长发，蓝色眼睛，戴着眼镜。性格：聪明好奇，勇敢坚定，对科学充满热情。动机：阻止军阀利用她的发明危害世界。

3. 军阀扎尔（反派）：45岁，独裁者，冷酷无情。外貌特征：高大魁梧，脸上有一道长刀疤，穿着黑色军装。性格：残忍暴虐，权力欲极强，视人命如草芥。动机：统治整个世界，消灭所有反抗者。
```

### 步骤4：扩展摘要为一页大纲

**操作指南：**
- 将摘要扩展为更详细的大纲（约1页）
- 包含主要情节转折点和关键场景
- 明确故事的起承转合

**示例输出：**
```
一页大纲：

1. 开场：凯德在酒吧接下寻找失踪科学家艾莉亚的任务，只知道她被军阀的人绑架了。

2. 发展：凯德追踪到艾莉亚的下落，在救她的过程中，他开始出现记忆碎片，看到自己曾经为军阀工作的画面。

3. 转折：艾莉亚告诉凯德，她发明了一种可以控制人类思想的装置，军阀想要利用它来统治世界。凯德意识到自己曾经参与过这个项目的开发。

4. 危机：军阀的军队包围了他们的藏身之处，凯德必须保护艾莉亚并阻止装置落入军阀手中。

5. 高潮：凯德与军阀的得力助手——他昔日的战友对决，在战斗中恢复了全部记忆，想起了自己当初是如何反抗军阀并被消除记忆的。

6. 结局：凯德击败了敌人，销毁了装置，带着艾莉亚开始了新的生活，同时也开始面对自己的过去。
```

### 步骤5：为每个角色创建更详细的背景

**操作指南：**
- 为每个主要角色创建更详细的背景故事
- 包含角色的成长经历、重要事件、人际关系和内心冲突
- 探索角色的动机和行为背后的原因

**示例输出：**
```
角色详细背景：

凯德：
- 童年：在孤儿院长大，从未见过父母。
- 青少年：加入军阀的军队，因表现出色被选为特种部队成员。
- 成年早期：成为军阀的顶级杀手，执行了许多残酷的任务。
- 关键事件：在一次任务中，他发现军阀计划用思想控制装置奴役民众，于是试图销毁装置，结果被抓住并消除了记忆。
- 内心冲突：他既想知道自己的过去，又害怕面对自己曾经的罪行。

艾莉亚：
- 童年：在一个科学家家庭长大，从小就对发明创造充满兴趣。
- 青少年：以优异的成绩考入顶尖大学，专攻神经科学。
- 成年早期：被军阀强迫为其工作，开发思想控制装置。
- 关键事件：她发现军阀的真正意图后，决定销毁装置并逃跑，结果被绑架。
- 内心冲突：她为自己参与开发危害人类的装置而感到内疚，同时又希望利用自己的知识造福社会。
```

### 步骤6：扩展大纲为四页

**操作指南：**
- 将一页大纲扩展为更详细的四页大纲
- 为每个主要情节添加更多细节和次要情节
- 明确场景之间的过渡和角色发展

**示例输出：**
```
四页大纲：

第一部分：任务接取
- 场景1：凯德在酒吧遇到神秘客户，接下寻找艾莉亚的任务。
- 场景2：凯德调查艾莉亚的背景，发现她是一位著名的科学家。
- 场景3：凯德追踪到军阀的据点，开始制定 rescue 计划。

第二部分：记忆觉醒
- 场景4：凯德潜入据点，与守卫发生冲突，开始出现记忆碎片。
- 场景5：凯德找到艾莉亚，她告诉凯德关于思想控制装置的事情。
- 场景6：两人逃离据点，在逃亡过程中，凯德的记忆逐渐恢复。

第三部分：真相揭露
- 场景7：凯德和艾莉亚找到一个安全的藏身之处，艾莉亚解释装置的工作原理。
- 场景8：凯德回忆起自己曾经参与装置开发的经历，以及他反抗军阀的原因。
- 场景9：军阀的军队发现了他们的藏身之处，开始围攻。

第四部分：最终对决
- 场景10：凯德和艾莉亚与军阀的军队展开激战，艾莉亚试图销毁装置。
- 场景11：凯德与昔日的战友对决，在战斗中恢复了全部记忆。
- 场景12：凯德击败敌人，销毁了装置，带着艾莉亚离开，开始新的生活。
```

### 步骤7：为每个场景创建简介

**操作指南：**
- 为大纲中的每个场景创建简短的简介
- 包含场景的地点、时间、主要事件和角色目标
- 明确场景在整个故事中的作用

**示例输出：**
```
场景简介：

场景1：酒吧接取任务
- 地点：破旧的酒吧，烟雾缭绕，灯光昏暗。
- 时间：晚上10点。
- 主要事件：神秘客户找到凯德，提供丰厚报酬让他寻找失踪的科学家艾莉亚。
- 角色目标：凯德需要决定是否接下任务，神秘客户希望凯德能尽快找到艾莉亚。
- 作用：故事的开端，引入任务和主要角色。

场景2：调查背景
- 地点：凯德的公寓，杂乱但有序，墙上贴满了各种任务资料。
- 时间：第二天上午。
- 主要事件：凯德通过网络和线人调查艾莉亚的背景，发现她是神经科学领域的专家。
- 角色目标：凯德需要了解艾莉亚的身份和可能的下落。
- 作用：提供背景信息，推进情节发展。
```

### 步骤8：开始正式写作

**操作指南：**
- 根据场景简介开始逐章写作
- 保持一致的叙述风格和角色声音
- 关注细节描写和情感表达
- 定期回顾大纲，确保情节连贯

**示例输出：**
```
小说开头：

烟雾在破旧的酒吧里缭绕，霓虹灯的蓝光映在吧台的玻璃上。凯德坐在角落的位置，面前放着一杯威士忌，手指无意识地敲击着桌面。他不喜欢这种地方，但这里是接任务的最佳地点。

一个穿着黑色西装的男人走了进来，环顾四周后，径直走向凯德的桌子。"你是凯德？"男人的声音低沉，带有一丝紧张。

凯德抬头看了他一眼，点了点头。"取决于你要什么。"

男人坐下来，从口袋里掏出一个信封，推到凯德面前。"我需要你找到一个女人，艾莉亚·陈。她是一位科学家，被军阀的人绑架了。"

凯德打开信封，里面是一张艾莉亚的照片和一叠现金。"为什么找我？"

"因为你是最好的。"男人的眼神坚定，"而且，她可能知道一些关于你过去的事情。"

凯德的手指突然停住了。关于他过去的事情？这是他一直在寻找的。他抬头看着男人，"多少钱？"

"足够让你下半辈子不用工作。"

凯德笑了笑，把现金收进口袋。"成交。"
```

## 使用建议

1. **循序渐进**：按照八个步骤依次进行，不要跳过任何步骤。
2. **灵活调整**：根据自己的写作风格和故事需求，适当调整每个步骤的内容。
3. **反复修改**：在每个步骤完成后，回顾并修改之前的内容，确保逻辑连贯。
4. **保持热情**：写作过程中遇到困难时，回顾你的创意和角色，重新激发写作热情。
5. **寻求反馈**：在适当的时候，向他人寻求反馈，改进你的故事。

## 用户干预机制

为了确保小说符合用户需求，在每个步骤中都可以进行用户干预。以下是具体的干预方法和流程：

### 干预流程

1. **步骤完成后**：AI会生成每个步骤的内容并呈现给用户
2. **用户审核**：用户查看生成的内容，评估是否符合自己的需求
3. **用户修改**：用户可以直接在生成的内容上进行修改，调整任何不满意的部分
4. **确认继续**：用户确认修改后的内容，然后进入下一个步骤

### 具体操作方式

- **直接编辑**：用户可以直接修改AI生成的文本内容，包括添加、删除或修改任何部分
- **保留满意部分**：用户可以只修改不满意的部分，保留满意的内容
- **微调调整**：用户可以进行小范围的调整，如修改角色名字、调整场景细节等
- **结构调整**：用户可以调整内容的结构，如重新组织大纲顺序、调整角色关系等

### 审核机制

- **逐步骤审核**：每个步骤完成后都会暂停，等待用户审核和修改
- **即时反馈**：用户可以在任何时候提供反馈，AI会立即响应
- **无强制重新生成**：用户修改后不需要AI重新生成，修改后的内容直接作为该步骤的最终结果
- **灵活回退**：用户可以随时回到之前的步骤进行修改，确保整个故事的连贯性

### 具体干预方法

#### 步骤1：一句话概括故事
- **用户干预点**：故事概括的各个元素
- **干预方式**：
  - "我希望主角是一位年轻的女医生，而不是赏金猎人"
  - "冲突应该是她发现医院的秘密实验，而不是被追杀"
  - "灾难元素可以是病毒爆发，而不是军阀统治"

#### 步骤2：扩展为一段摘要
- **用户干预点**：摘要的结构和内容
- **干预方式**：
  - "我希望摘要更强调主角的内心挣扎"
  - "可以增加一个配角，比如主角的同事"
  - "结局应该更开放一些，留下续集的可能性"

#### 步骤3：为主要角色创建简介
- **用户干预点**：角色的各个属性
- **干预方式**：
  - "主角的名字改为林小雨，年龄26岁"
  - "她的性格应该更坚韧，有强烈的正义感"
  - "反派可以是医院的院长，表面和蔼实则冷酷"

#### 步骤4：扩展摘要为一页大纲
- **用户干预点**：大纲的结构和情节
- **干预方式**：
  - "我希望增加一个主角发现线索的场景"
  - "高潮部分应该更紧张，加入追逐戏"
  - "可以调整情节顺序，先展示主角的日常生活"

#### 步骤5：为每个角色创建更详细的背景
- **用户干预点**：角色的背景故事
- **干预方式**：
  - "主角的童年经历可以增加一个关于母亲的情节"
  - "反派的动机应该更复杂，不是单纯的坏"
  - "配角的背景可以与主角有联系，增加戏剧冲突"

#### 步骤6：扩展大纲为四页
- **用户干预点**：详细大纲的内容和结构
- **干预方式**：
  - "我希望增加更多关于医院内部的场景"
  - "可以添加一个主角与反派的对话场景"
  - "调整场景顺序，使情节更紧凑"

#### 步骤7：为每个场景创建简介
- **用户干预点**：场景的各个元素
- **干预方式**：
  - "场景1的地点可以改为雨夜的医院急诊室"
  - "场景3的时间设置为凌晨3点，增加紧张感"
  - "场景5的主要事件可以加入一个意外的发现"

#### 步骤8：开始正式写作
- **用户干预点**：具体的文本内容
- **干预方式**：
  - "这段描写可以更细腻，突出主角的心理活动"
  - "对话部分可以更自然，符合角色性格"
  - "场景转换可以更流畅，加入过渡描写"

### 干预示例

**示例1：修改角色设定**
- AI生成："凯德（主角）：35岁，赏金猎人，身手敏捷，擅长各种武器。"
- 用户干预："我希望主角是一位28岁的女记者，擅长调查和推理，性格坚韧但有同情心。"
- AI调整："林小雨（主角）：28岁，记者，擅长调查和推理，性格坚韧但有同情心。"

**示例2：调整情节发展**
- AI生成："场景4：凯德潜入据点，与守卫发生冲突，开始出现记忆碎片。"
- 用户干预："我希望场景4改为主角在医院夜班时，发现异常的病人病例，开始怀疑医院有秘密。"
- AI调整："场景4：林小雨在医院夜班时，发现异常的病人病例，开始怀疑医院有秘密。"

**示例3：修改写作风格**
- AI生成："烟雾在破旧的酒吧里缭绕，霓虹灯的蓝光映在吧台的玻璃上。"
- 用户干预："我希望风格更写实，描写更细腻，突出医院的紧张氛围。"
- AI调整："消毒水的气味在医院走廊里弥漫，白色的荧光灯照在冰冷的瓷砖地面上，反射出刺眼的光。"

### 整体干预建议
1. **明确具体**：提供具体的修改意见，而不是模糊的反馈
2. **保持连贯**：修改时考虑与之前内容的连贯性
3. **风格一致**：确保修改后的内容与整体风格保持一致
4. **循序渐进**：在每个步骤都进行适当的干预，而不是等到最后才大量修改
5. **开放沟通**：随时提出新的想法和要求，AI会根据你的反馈进行调整

## 常见问题

### Q: 雪花写作法适合所有类型的小说吗？
A: 是的，雪花写作法适用于各种类型的小说，包括科幻、奇幻、悬疑、 romance等。

### Q: 如果我在某个步骤卡住了怎么办？
A: 可以暂时跳过这个步骤，先进行下一个步骤，然后再回来解决。或者尝试自由写作，让创意自然流动。

### Q: 雪花写作法需要多长时间完成？
A: 这取决于你的故事复杂度和写作速度。一般来说，完成前七个步骤可能需要几周到几个月的时间，而正式写作则可能需要更长时间。

### Q: 我可以在写作过程中修改大纲吗？
A: 当然可以。雪花写作法是一个灵活的框架，不是僵硬的规则。随着你对故事和角色的理解加深，修改大纲是正常的。

## 总结

雪花写作法是一种结构化的小说创作方法，通过从简单到复杂的逐步扩展，帮助你构建一个完整、连贯的故事。它不仅可以帮助你克服写作障碍，还可以让你的故事更加丰满和有深度。无论你是新手还是经验丰富的作家，雪花写作法都可以成为你创作过程中的有力工具。
